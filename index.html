What is?

Yupi is an OSP Habbo Emulator, not created for production usages. Made for cool guys. Not for Production Usage and not intendend to make money with it. You're free to do what u want, breaching Copyrights is your responsability.

Based in?

Butterfly Emulator, originally created by Martinmine. Actually Based in a Butterfly's Edition called Azure Emulator. Observation: In other terms the code wasn't created from scratch by the collaborators, but based in Butterfly Emulator.
